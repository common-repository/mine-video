== Description ==

* 推荐使用新版插件 [Mine 云点播](https://wordpress.org/plugins/mine-cloudvod/)，老版(此)插件不再升级维护。

* 新版插件支持经典编辑器和古腾堡编辑器，功能和兼容性更强，适用于各种视频播放和音频播放的站点。

* 新版插件包含此插件所有免费功能和更多更实用功能，强烈推荐使用新版插件，感谢使用和支持。
